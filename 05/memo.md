## List(리스트)  
다른 언어에서의 배열이다.  
형식 :   
ex. fruit = ['사과', '베', '딸기', '포도']  
    print(fruit)
    결과값 : ['사과', '베', '딸기', '포도']  
    
### < 항목 추가하기 >  
fruit.append('귤')  
print(fruit)    
결과값 : ['사과', '베', '딸기', '포도', '귤']   

### < 항목 확인하기 >  
if '딸기' in fruit :   
  print("딸기가 있네요 ^^")  
 결과값 : 딸기가 있네요 ^^     --> if 항목 in 리스트 : 리스트 안에 항목이 있으면 true를 반환한다.
